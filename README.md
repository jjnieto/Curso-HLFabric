# Curso HLFabric

Material público del curso de **Administración y desarrollo con Hyperledger Fabric V2**.

Aquí encontrarás las slides, documentación y ejercicios organizados por módulos. El contenido se sube progresivamente conforme avanzan las clases.

## Estructura

```
Curso HLFabric/
├── Modulo 1 - Introduccion a Blockchain y Fabric/
├── Modulo 2 - Desarrollo de proyectos con Fabric/
├── Modulo 3 - Vision empresarial/
├── Modulo 4 - Tokens y Smart Contracts/
├── Modulo 5 - Identidad Digital/
└── Modulo 6 - Proyecto final (FidelityChain)/
```

## Cómo usar este repositorio

1. Clónalo en tu máquina:
   ```bash
   git clone https://github.com/jjnieto/Curso-HLFabric.git
   ```
2. Sigue las clases y los documentos del módulo correspondiente.
3. Las soluciones a los ejercicios se publican **después** de la sesión presencial — intenta resolverlos por tu cuenta primero.

## Material adicional

- Las **presentaciones** en `.pptx` se distribuyen también en PDF para facilitar su consulta sin PowerPoint.
- Los **ejercicios prácticos** incluyen guías paso a paso para montar redes, desplegar chaincodes y desarrollar aplicaciones cliente.
- El **proyecto final** (Módulo 6) está en su propio directorio con código fuente listo para compilar.

## Soporte

- Dudas durante las clases: pregunta directamente al instructor.
- Dudas fuera de clase: usa el sistema habitual del curso (Slack, email, foro).
